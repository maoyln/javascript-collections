addEventListener 是 JavaScript 中用于为 DOM 元素添加事件监听器的方法。 它允许开发者将一个或多个事件处理函数附加到元素上，当特定事件发生时，这些处理函数会被调用。 addEventListener 提供了一种灵活且强大的方式来处理用户交互事件，如点击、键盘输入、鼠标移动等。
1. 语法 element.addEventListener(event, handler, [options]); 参数： element: 要添加事件监听器的 DOM 元素。 event: 事件的类型，以字符串形式表示，如 "click"、"keydown"、"mouseover" 等。 handler: 当事件触发时要执行的函数（即事件处理程序）。 options (可选): 一个对象，包含监听器特性的配置。或使用 useCapture 布尔值，指定事件是在捕获阶段还是冒泡阶段触发。
2. 常见参数 事件类型 (event)：事件的类型，以字符串形式表示，如 "click"、"keydown"、"mouseover" 等。 事件处理程序 (handler)：事件触发时要执行的函数。这个函数可以接收一个事件对象参数，该参数包含事件的详细信息。
2. 常见参数 事件类型 (event)：事件的类型，以字符串形式表示，如 "click"、"keydown"、"mouseover" 等。 事件处理程序 (handler)：事件触发时要执行的函数。这个函数可以接收一个事件对象参数，该参数包含事件的详细信息。
3. 常见选项 useCapture: 一个布尔值，默认为 false，表示事件处理程序是在事件冒泡阶段执行（false）还是在捕获阶段执行（true）。 once: 一个布尔值，表示事件处理程序是否只执行一次，执行后自动移除监听器。 passive: 一个布尔值，表示事件处理程序永远不会调用 preventDefault()，如果调用了，将被忽略。
